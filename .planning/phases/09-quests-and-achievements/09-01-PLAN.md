   - started_date: date (when the quest was assigned)

7. **QuestCompletion(BaseModel)** (for pending notification):

   - bits_reward: int

   - xp_reward: int

   - id: str (e.g., "battle_initiate")

   - tiers: list[AchievementTier] (Bronze, Silver, Gold — always 3)
   - stat_key: str (maps to PlayerProfile field name, e.g., "battles_won")

10. **AchievementUnlock(BaseModel)** (for pending notification):
    - achievement_name: str
    - tier_label: str
    - xp_reward: int
    - bits_reward: int
