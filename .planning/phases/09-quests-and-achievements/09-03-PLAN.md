**check_achievements(state: GameState) -> None:**
For each achievement in ACHIEVEMENT_CATALOG:
- value = get_stat_value(state, achievement.stat_key)
- unlocked = state.achievement_state.get(achievement.id, [])
- For each tier in achievement.tiers:
  - If value >= tier.threshold AND tier.label not in unlocked:
    - Initialize list if needed: state.achievement_state.setdefault(achievement.id, [])
    - Append tier.label to state.achievement_state[achievement.id]
    - state.player.xp += tier.xp_reward
    - state.player.currency += tier.bits_reward
    - Append AchievementUnlock(achievement_name=achievement.name, tier_label=tier.label, xp_reward=tier.xp_reward, bits_reward=tier.bits_reward) to state.pending_achievement_unlocks
