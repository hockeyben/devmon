1. **render_quest_list(quests: list[ActiveQuest], theme: dict, console: Console | None = None) -> Panel:**
   Surface 1 from UI-SPEC. Full-width panel titled "Active Quests".

   For each quest, render a row:
   ```
     [n] {quest.name}                        {difficulty badge}
         Progress: {BarColumn 20-wide}  {current}/{target}
         Reward:   {xp} XP  +{bits} Bits  {item or empty}
   ```

   - Quest name in "white"
   - Difficulty badge: easy="white", medium="bright_blue", hard="magenta" (from UI-SPEC color section)
   - Progress bar using Rich Progress with BarColumn(bar_width=20) and MofNCompleteColumn()
   - For multi-criteria quests (mixed), show each criterion on its own progress line
   - Reward line in "dim white" for labels, "white" for values
   - Category divider between coding/game/mixed sections: "  " + "-" * 38 in "dim white"
   - If quest slots are empty (completed, not yet refreshed): show "(empty -- refreshes tomorrow)" in "dim white"
   - If zero quests total (Surface 5): "No active quests. Code something today to unlock your first quest." in "dim white"
   - Panel border: theme["border"], box.ROUNDED, expand=True

   Use Rich Text() with .append() for body construction (matches all existing render modules).

   For the progress bar, since Rich Progress is designed for live display, use a simpler approach: build a text-based bar like the status.py xp_bar pattern, or use `"[{filled}{'.' * empty}]"` format. Actually, follow the UI-SPEC exactly: use `Progress(BarColumn(bar_width=20), MofNCompleteColumn())` but render it to a string via `console.file = StringIO()`. Or simpler: compute progress inline as `f"{current}/{target}"` with a visual bar character string.

   Simplest approach matching existing codebase: render progress as text with filled/empty block characters:
   ```python
   filled = int(20 * min(current, target) / max(target, 1))
   bar = "[" + "=" * filled + " " * (20 - filled) + "]"
   ```

   Then `f"  Progress: {bar}  {current}/{target}"` styled with theme.

**Part B: Create src/devmon/commands/quests.py**

Typer app for `devmon quests` command. Follow the pattern from commands/shop.py or commands/items.py.

```python
"""devmon quests -- view active quests and progress.

Requirements: QUST-05, CLI-07
"""
import typer
from rich.console import Console

app = typer.Typer()

@app.callback(invoke_without_command=True)
def quests_command() -> None:
    """View active quests and progress."""
    from devmon.config.loader import load_config
    from devmon.config.defaults import DEFAULT_CONFIG
    from devmon.persistence.save import load as load_state
    from devmon.render.themes import get_theme
    from devmon.render.quests import render_quest_list

    try:
        config = load_config()
    except Exception:
        config = DEFAULT_CONFIG

    state = load_state()
    if state is None:
        typer.echo("No save file found. Run some commands first!")
        raise typer.Exit(1)

    theme = get_theme(config.get("ui", {}).get("theme", "neon"))
    console = Console()
    panel = render_quest_list(state.active_quests, theme)
    console.print(panel)
```
