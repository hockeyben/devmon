```python
QuestDifficulty = Literal["easy", "medium", "hard"]
QuestCategory = Literal["coding", "game", "mixed"]

class QuestCriterion(BaseModel):
    type: str  # "total_commands", "battles_won", "git_commits", "creatures_captured", etc.
    target: int
    current: int = 0

class QuestTemplate(BaseModel):
    id: str
    name: str
    description: str
    difficulty: QuestDifficulty
    category: QuestCategory
    criteria: list[QuestCriterion]
    xp_reward: int
    bits_reward: int
    item_reward_id: Optional[str] = None

class ActiveQuest(BaseModel):
    id: str; name: str; description: str; difficulty: QuestDifficulty
    category: QuestCategory; criteria: list[QuestCriterion]
    xp_reward: int; bits_reward: int; item_reward_id: Optional[str] = None
    started_date: date

class QuestCompletion(BaseModel):
    quest_name: str; xp_reward: int; bits_reward: int; item_reward: Optional[str] = None
```

**Functions:**

1. `daily_quest_refresh(state, today=None)`:
   - If today is None, use date.today()
   - If state.quest_last_refresh_date == today, return immediately (Pitfall 2)
   - Count active quests by category: coding_count, game_count, mixed_count
   - Need = {coding: 2 - coding_count, game: 2 - game_count, mixed: 1 - mixed_count}
   - For each category with need > 0: filter QUEST_TEMPLATES by category, exclude templates whose id matches any active quest id, randomly sample min(need, available) templates
   - For each selected template: create ActiveQuest by copying fields + started_date=today, append to state.active_quests
   - Set state.quest_last_refresh_date = today
   - Import random at top of module for sampling

2. `grant_quest_reward(state, quest) -> QuestCompletion`:
   - state.player.xp += quest.xp_reward
   - state.player.currency += quest.bits_reward
   - item_name = None
   - If quest.item_reward_id is not None: state.inventory[quest.item_reward_id] = state.inventory.get(quest.item_reward_id, 0) + 1; item_name = quest.item_reward_id
   - Return QuestCompletion(quest_name=quest.name, xp_reward=quest.xp_reward, bits_reward=quest.bits_reward, item_reward=item_name)

3. `check_quest_completions(state, config)`:
   - completed = []
   - remaining = []
   - For each quest in state.active_quests: if all(c.current >= c.target for c in quest.criteria): completed.append(quest) else: remaining.append(quest)
   - For each quest in completed: completion = grant_quest_reward(state, quest); state.pending_quest_completions.append(completion)
   - state.active_quests = remaining
   - If completed and len(remaining) == 0: state.daily_bonus_pending = True; state.player.xp += 100; state.player.currency += 50

4. `update_coding_quest_progress(state, events)`:
   - Count from events: cmd_count = sum(1 for e in events if e.get("type","cmd")=="cmd" and e.get("exit",1)==0), git_count = sum(1 for e in events if e.get("type")=="git_commit"), test_count = sum(1 for e in events if e.get("type")=="test_pass")
   - For each active quest, for each criterion: if type=="total_commands": current += cmd_count; if type=="git_commits": current += git_count; if type=="test_passes": current += test_count

5. `update_game_quest_progress(state, event_type)`:
   - Map: {"battle_win": "battles_won", "creature_captured": "creatures_captured", "rare_capture": "rare_captures", "encounter_seen": "encounters_seen"}
   - criterion_type = map.get(event_type)
   - If criterion_type: for each active quest, for each criterion matching type: current += 1
