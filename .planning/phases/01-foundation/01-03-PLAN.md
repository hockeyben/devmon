Write src/devmon/config/loader.py:

Exports two functions:

load_config() -> dict[str, Any]:
- Resolves config path: os.environ.get("DEVMON_HOME") / "config.toml" if set, else platformdirs.user_data_dir("devmon", "devmon") / "config.toml"
- If path does not exist: return deep copy of DEFAULT_CONFIG
- If path exists: open in "rb" mode, parse with tomllib.load(), then deep-merge DEFAULT_CONFIG with user values (user values win, missing keys filled from defaults)
- Returns merged dict

save_config(cfg: dict[str, Any]) -> None:
- Resolves config path same as above
- Creates parent directories with mkdir(parents=True, exist_ok=True)
- Opens in "wb" mode and writes with tomli_w.dump(cfg, f)

Helper functions (private):
- _config_path() -> pathlib.Path — resolves path with DEVMON_HOME check
- _deep_copy(d: dict) -> dict — shallow-recursive copy
- _deep_merge(base: dict, override: dict) -> dict — merge override into base; nested dicts are recursively merged

Imports required: os, pathlib, tomllib (stdlib), tomli_w, typing.Any, devmon.config.defaults.DEFAULT_CONFIG
