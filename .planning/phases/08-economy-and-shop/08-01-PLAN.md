| Boundary | Description |
|----------|-------------|
| JSON data files -> item_loader | User-editable item JSON could contain malformed data |
| Save file -> GameState | Inventory dict could have negative values or unknown item IDs |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-08-01 | T (Tampering) | item JSON data files | mitigate | Pydantic model_validate() rejects invalid data at load time; price Field(ge=0) prevents negative prices |
| T-08-02 | T (Tampering) | inventory dict in save | accept | Inventory is local save file — user can edit their own save. No server-side validation needed for local CLI game |
| T-08-03 | D (DoS) | consume_item negative qty | mitigate | consume_item checks `current >= qty` before deduction; qty parameter validated as positive |
</threat_model>
