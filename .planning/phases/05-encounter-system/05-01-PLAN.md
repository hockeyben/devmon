## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| JSON creature data -> model validation | Untrusted JSON files (user-editable) loaded into Pydantic models |
| Save file -> GameState load | Save file could be hand-edited or from a future version |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | creature JSON allowed_rarities | mitigate | Pydantic validates allowed_rarities contains only valid CreatureRarity literals; invalid values rejected at load time |
| T-05-02 | Tampering | save file encounter_queue | mitigate | EncounterEntry is a Pydantic model; malformed encounter_queue in save file rejected by model_validate() |
| T-05-03 | Denial of Service | encounter_history unbounded | mitigate | Cap encounter_history list length in engine (Plan 02); model itself is a list but engine enforces max size |
</threat_model>
