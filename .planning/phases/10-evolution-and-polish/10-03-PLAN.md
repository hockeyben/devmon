<objective>
Add narrow terminal adaptation (width < 40) to all render modules that display ASCII art or wide layouts, per UI-06 and D-08.

Purpose: The game should degrade gracefully in narrow terminals (tmux panes, split VS Code terminals) without rendering errors or truncation. ASCII art is hidden entirely when width < 40. HP bars compress to width=10.

Output: All render functions that show ASCII art accept a `narrow: bool = False` parameter. All call sites detect `console.width < 40` and pass the result.
</objective>
