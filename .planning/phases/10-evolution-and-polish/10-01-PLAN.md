    Create src/devmon/engine/evolution_engine.py as a pure domain logic module following the battle_engine.py pattern:
    - `from __future__ import annotations` + TYPE_CHECKING guard for model imports
    - No Rich, no Typer, no persistence imports

    Functions:
    1. `check_evolution_ready(owned: "OwnedCreature", template: "CreatureTemplate") -> bool`:
       - Returns False if template.evolves_to is None
       - Returns False if template.evolution_level_threshold is None
       - Returns False if owned.evolution_declined is True
       - Returns True if owned.level >= template.evolution_level_threshold

    2. `check_condition_evolution(owned: "OwnedCreature", template: "CreatureTemplate") -> bool`:
       - Returns False if template.evolves_to is None or template.evolution_condition is None
       - Reads condition dict: if type == "battles_won", check owned.battles_won_with >= count
       - Returns False for unknown condition types

    3. `apply_evolution(owned: "OwnedCreature", evolved_template_id: str) -> None`:
       - Mutates owned.template_id = evolved_template_id
       - Resets owned.evolution_declined = False
       - Resets owned.battles_won_with = 0
       - Sets owned.current_hp = None (force recompute from new template)

    4. `clear_evolution_declined_on_level_up(owned: "OwnedCreature") -> None`:
       - Sets owned.evolution_declined = False
       - Called after apply_creature_xp returns True (leveled up), before checking evolution readiness (per D-02, Pitfall 1)

    Then update tests/test_evolution.py: remove xfail markers from engine test stubs and implement the test bodies using direct function calls with mock OwnedCreature/CreatureTemplate instances.
