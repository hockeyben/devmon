Module docstring: "Creature panel rendering for Rich terminal UI. Pure render module — no I/O, no state mutation."

ARCHITECTURE rule: This module imports from `devmon.models.creature` (types only) and `devmon.render.themes` (colors). It must NOT import from commands/, engine/, config/, or persistence/.

Implement `render_creature_panel(template: CreatureTemplate, console: Console, theme: dict[str, str] | None = None) -> None`:

1. Import `Panel`, `Text` from `rich.panel` and `rich.text`. Import `box` from `rich`. Import `RARITY_COLORS` from `devmon.render.themes`. Import `get_theme` from `devmon.render.themes`.
2. Look up `border_color = RARITY_COLORS.get(template.rarity, "white")`.
3. If `theme` is None, use `get_theme("neon")` as default.
4. Build art `Text` object: iterate `template.ascii_art`, append each line with `style=template.primary_color`. Add newline between lines (not after last).
5. Build stats text block using two-column layout with aligned spacing:
   ```
   HP     {base_hp:<6}  Type    {type}
   ATK    {base_attack:<6}  SPD     {base_speed}
   DEF    {base_defense:<6}  Capture {capture_rate:.0%}
   ```
   Stat keys styled with `theme["stat_key"]`, stat values with `theme["stat_value"]`.
6. Build flavor text: `Text(template.flavor_text, style="dim white")`.
7. Combine art + blank line + stats + blank line + flavor into a single `Text` group.
8. Wrap in `Panel(group, title=..., subtitle=..., border_style=border_color, box=box.ROUNDED, expand=False)`.
   - `title=f"[{border_color}]{template.name}[/{border_color}]"`
   - `subtitle=f"[dim]{template.rarity.title()} - {template.type}[/dim]"`
9. `console.print(panel)`.

Also export `RARITY_COLORS` re-import for convenience (or just reference from themes.py).
